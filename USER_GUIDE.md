## Getting Started

Welcome to **Task Insight DMS** (Document Management System)! This guide will help you get started with managing your documents efficiently.

### System Requirements

- **Web Browser**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **Internet Connection**: Stable broadband connection
- **Screen Resolution**: Minimum 1280x720 pixels

### First-Time Setup

1. Receive an **invitation code** from your organization administrator
2. Navigate to the Task Insight DMS website
3. Click **"Create Account"**
4. Follow the registration process
5. Verify your email address
6. Login and start using the system

---

## User Registration

### Joining an Organization

If you've been invited to join an existing organization:

1. **Navigate to Registration Page**
   - Go to the Task Insight DMS login page
   - Click on the **"Create Account"** tab

2. **Select "Join Organization"**
   - Choose the "Join Organization" option
   - Enter your invitation code

3. **Fill in Your Details**
   - **First Name**: Your first name
   - **Last Name**: Your last name
   - **Email**: Your email address
   - **Password**: Choose a strong password (minimum 6 characters)
   - **Confirm Password**: Re-enter your password

4. **Submit Registration**
   - Review your information
   - Click **"Join Organization"**
   - Check your email for verification link

5. **Verify Email**
   - Open the verification email
   - Click the verification link
   - Your account is now active!

### Password Requirements

Your password must meet these criteria:
- Minimum 6 characters long
- Recommended: Include uppercase, lowercase, numbers, and special characters
- Avoid common passwords or personal information

## File Management

### Uploading Files

#### Method 1: Click Upload Button

1. Click the **"Upload File"** button in the top bar
2. Click **"Select File"** or drag and drop
3. Choose your file from your computer
4. Select destination folder (optional)
5. Add description (optional)
6. Click **"Upload"**

#### Method 2: Drag and Drop

1. Open the folder where you want to upload
2. Drag file(s) from your computer
3. Drop them into the dashboard
4. Upload begins automatically

#### Supported File Types

- **Documents**: PDF, DOC, DOCX, TXT, RTF
- **Spreadsheets**: XLS, XLSX, CSV
- **Presentations**: PPT, PPTX
- **Images**: JPG, JPEG, PNG, GIF, BMP
- **Videos**: MP4, AVI, MOV, WMV
- **Archives**: ZIP, RAR, 7Z
- **Other**: Most common file formats

#### File Size Limits

- **Maximum file size**: 10 MB (varies by organization)
- **Storage quota**: Based on organization plan

### Viewing Files

#### Preview Files

1. Click on a file card
2. Preview modal opens
3. View file details:
   - File name
   - File size
   - Upload date
   - Uploader name
   - Description

#### Download Files

**Option 1**: From preview modal
- Click **"Download"** button

**Option 2**: From file menu
- Click three-dot menu (‚ãÆ) on file card
- Select **"Download"**

### File Operations

#### Rename File

1. Click three-dot menu (‚ãÆ) on file card
2. Select **"Rename"**
3. Enter new file name
4. Click **"Save"**

#### Move File

1. Click three-dot menu (‚ãÆ) on file card
2. Select **"Move"**
3. Choose destination folder
4. Click **"Move"**

#### Copy File

1. Click three-dot menu (‚ãÆ) on file card
2. Select **"Copy"**
3. Choose destination folder
4. Click **"Copy"**

#### Delete File

1. Click three-dot menu (‚ãÆ) on file card
2. Select **"Delete"**

1. Go to **"Trash"** in sidebar

1. Go to **"Trash"** in sidebar

## File Sharing

### Share a File

1. Click on file to open preview
2. Click **"Share"** button
3. Configure sharing options:
   - **Permission Level**: View, Comment, or Edit
   - **Expiration Date**: Set when link expires
   - **Password Protection**: Optional password
   - **Share with Email**: Enter recipient email (optional)

4. Click **"Generate Link"**
5. Copy the share link
6. Send link to recipients

### Permission Levels

- **View**: Can only view and download the file
- **Comment**: Can view, download, and add comments
- **Edit**: Can view, download, comment, and modify

### Managing Shares

#### View Active Shares

1. Click on file
2. Click **"Share"** button
3. See list of active shares

#### Revoke Share

1. Open file sharing dialog
2. Find the share link
3. Click **"Revoke"** or **"Delete"**
4. Confirm revocation

### Accessing Shared Files

When someone shares a file with you:

1. Click the share link
2. Enter password (if required)
3. View or download the file
4. Perform actions based on permission level

## Troubleshooting

### Common Issues

#### Cannot Login

**Problem**: Invalid email or password

**Solutions**:
1. Check for typos in email
2. Ensure Caps Lock is off
3. Use "Forgot Password" to reset
4. Clear browser cache and cookies
5. Try a different browser

---

#### File Upload Fails

**Problem**: Upload doesn't complete

**Solutions**:
1. Check file size (must be under limit)
2. Verify internet connection
3. Try a different browser
4. Check if file type is supported
5. Clear browser cache
6. Contact support if issue persists

---

#### Cannot See Shared Files

**Problem**: Shared link doesn't work

**Solutions**:
1. Check if link has expired
2. Verify you have permission
3. Enter correct password (if required)
4. Check internet connection
5. Ask sender to generate new link

---

#### Storage Full

**Problem**: Cannot upload more files

**Solutions**:
1. Delete unnecessary files
2. Empty trash folder
3. Remove duplicate files
4. Compress large files before upload
5. Contact admin to increase quota

---

#### Slow Performance

**Problem**: Dashboard loads slowly

**Solutions**:
1. Clear browser cache
2. Close unnecessary browser tabs
3. Check internet speed
4. Try different browser
5. Reduce number of files per folder
6. Use filters to limit displayed items

## FAQs

### General Questions

**Q: Is my data secure?**  
A: Yes! We use industry-standard encryption for data at rest and in transit. All file transfers use HTTPS/SSL.

**Q: Can I access files offline?**  
A: Currently, Task Insight DMS requires an internet connection. Offline access is planned for future releases.

**Q: What happens when I reach my storage limit?**  
A: You won't be able to upload new files until you free up space or your admin increases your quota.

**Q: Can I recover deleted files?**  
A: Yes, deleted files remain in Trash for 30 days and can be restored during this period.

### File Management

**Q: What's the maximum file size I can upload?**  
A: The default limit is 10 MB, but this may vary by organization. Check with your administrator.

**Q: Can I upload multiple files at once?**  
A: Yes, you can drag and drop multiple files or use the multi-select option in the upload dialog.

**Q: Are file versions tracked?**  
A: Yes, when you re-upload a file with the same name, previous versions are preserved.

### Sharing

**Q: Can I share folders?**  
A: Currently, only individual files can be shared. Folder sharing is coming in a future update.

**Q: Do shared links expire?**  
A: You can set an expiration date when creating a share link. Links without an expiration remain active until revoked.

**Q: Can I see who accessed my shared files?**  
A: Yes, in the sharing dialog you can see access count and last access time.

### Organization & Admin

**Q: How do I upgrade my organization's storage?**  
A: Contact your platform administrator or Task Insight support to discuss storage upgrade options.

**Q: Can I change my organization?**  
A: Organization changes require administrator approval. Contact your organization admin.

**Q: What's the difference between roles?**  
A:
- **Platform Owner**: Full system access, manages all organizations
- **Org Admin**: Manages users and settings within their organization
- **Member**: Standard user with file management capabilities

### Technical

**Q: Which browsers are supported?**  
A: Chrome, Firefox, Safari, and Edge (all latest versions). Internet Explorer is not supported.

**Q: Does Task Insight DMS work on mobile devices?**  
A: Yes, the interface is responsive and works on tablets and smartphones.

**Q: Can I integrate Task Insight DMS with other tools?**  
A: API integration is available. Check the API documentation for details.

## Best Practices

### File Organization

1. **Use descriptive file names** including dates
2. **Create folder structure** before uploading
3. **Use tags** for easy categorization
4. **Add descriptions** to important files
5. **Regular cleanup** of old/unused files

### Security

1. **Use strong passwords** (12+ characters)
2. **Enable two-factor authentication** (when available)
3. **Don't share passwords** for shared links
4. **Set expiration dates** for shares
5. **Review active shares** regularly
6. **Logout when using shared computers**

### Storage Management

1. **Compress files** before uploading
2. **Delete duplicates** regularly
3. **Empty trash** periodically
4. **Archive old files** to external storage
5. **Monitor storage usage** dashboard
